## 	实有人口数据上报
### 请求URL

```
http://[server]/personnelRecord/add
```

### 请求头部

```
以公共信息页面为准
```

### 请求方式

```
POST 
```

### 请求体类型

```
# 表单
Content-Type:application/x-www-form-urlencoded
```
